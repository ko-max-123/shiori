# 旅しおりメーカー MVP

APIなし・DBなし・ログインなしで動く、静的HTML版の旅行しおり作成ツールです。

## 使い方

1. ZIPを展開します。
2. `index.html` をブラウザで開きます。
3. 旅行タイトル、日程、予定、持ち物を入力します。
4. 左側の「しおり一覧」から、これまで作成したしおりを開いたり、新規しおりを作成したりできます。
5. 右側にしおりプレビューが表示されます。
6. PCでは「しおりのみ表示」、スマホでは上部または下部固定の「しおりを表示」から、入力欄を隠してしおりだけ確認できます。
7. 「印刷 / PDF保存」からPDF保存できます。
8. 印刷時は入力フォームを除外し、しおりプレビューに近いカラー付きレイアウトで出力されます。

- 3カラム構成（左：しおり一覧 / 中央：編集 / 右：しおりプレビュー）
- 保存済みしおり一覧

- 保存済みしおり一覧の表示
- 複数しおりの切り替え / 新規作成 / 削除
- タイトル背景画像の設定（アップロード / 解除）
- 日別スケジュール作成
- 予定カードのドラッグ＆ドロップ並べ替え
- 予定カテゴリ管理
- 予定ごとの費用入力
- 概算予算集計
- 持ち物チェックリスト
- しおりプレビュー
- カラー印刷 / PDF保存向けレイアウト
- PC向け「しおりのみ表示」モード
- スマホ向け上部・下部固定「しおりを表示」ボタン
- しおりのみ表示中の「編集に戻る」操作
- Markdownコピー / 保存
- JSON保存 / 読込
- localStorageによる自動保存
- このブラウザ内での複数しおり保存
- サンプルデータ読込

## 印刷 / PDF保存について

「印刷 / PDF保存」を押すと、入力フォームや操作ボタンは非表示になり、しおり部分だけが印刷対象になります。
Chrome / Edge で色が薄く出る場合は、印刷画面の詳細設定で「背景のグラフィック」を有効にしてください。

タイトル背景画像を設定した場合も、しおりカバーと印刷 / PDF保存の両方に反映されます。

## データ保存について

入力データとしおり一覧は、ブラウザの localStorage に保存されます。
外部サーバーには送信されません。

作成したしおりは編集画面のさらに左側にある「しおり一覧」に保存されます。
同じブラウザ内であれば、複数のしおりを切り替えて編集できます。

別PCや別ブラウザへ移したい場合は、「JSON保存」でデータを保存し、移動先で「JSON読込」を使ってください。
